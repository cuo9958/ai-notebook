# 笔记导出功能使用说明

## 功能概述
已在笔记编辑页面上方添加了导出下拉按钮,支持将笔记导出为以下格式:
- **Markdown (.md)**: 原始 Markdown 格式
- **HTML (.html)**: 带样式的 HTML 文件
- **PDF (.html)**: 通过 HTML 格式导出,可在浏览器中打印为 PDF

## 使用方法

1. **打开笔记**
   - 在左侧笔记树中选择要导出的笔记

2. **点击导出按钮**
   - 在笔记编辑区域上方的工具栏中,找到"导出"按钮(带有下载图标)
   - 点击后会显示下拉菜单

3. **选择导出格式**
   - 在下拉菜单中选择需要的导出格式:
     - 导出为 Markdown
     - 导出为 HTML
     - 导出为 PDF

4. **选择导出目录**
   - 选择格式后,会弹出目录选择对话框
   - 选择您想要保存导出文件的位置
   - 点击确认

5. **完成导出**
   - 文件会自动保存到所选目录
   - 文件名格式: `笔记名称_时间戳.扩展名`
   - 例如: `测试笔记_2026-04-29T17-30-45.md`

## 导出文件说明

### Markdown 格式
- 保留原始 Markdown 内容
- 适合后续编辑和版本控制

### HTML 格式
- 包含完整的 HTML 结构
- 内置样式,可直接在浏览器中打开
- 支持代码高亮、表格、引用等 Markdown 元素

### PDF 格式
- 实际导出为 HTML 格式
- 用户可以在浏览器中打开后使用打印功能保存为 PDF
- 这样可以保持最佳的排版效果

## 注意事项

1. **自动保存**: 导出前会自动保存当前笔记的未保存内容
2. **图片处理**: 
   - 本地上传的图片会正常显示
   - 外部图片链接会保留原始链接
3. **文件命名**: 自动添加时间戳避免文件名冲突
4. **目录权限**: 确保选择的导出目录有写入权限

## 技术实现

- 使用 Tauri 的 `@tauri-apps/plugin-dialog` 进行目录选择
- 使用 `@tauri-apps/plugin-fs` 进行文件写入
- HTML 导出使用 marked 库进行 Markdown 到 HTML 的转换
- 样式采用响应式设计,适配不同屏幕尺寸

## 故障排除

如果导出失败:
1. 检查是否选择了有效的导出目录
2. 确认对该目录有写入权限
3. 查看控制台日志获取详细错误信息
4. 确保笔记内容不为空
